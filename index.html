HTML

  • The header element contains information about the webpage
  • The src element contains the location and image/video
  • The body represents the visual content shown to the user.

CSS

    • A margin indicates how much space we want around the outside of an element.
    • A padding indicates how much space we want around the content inside an element.
    • assigning classes to certain elements.
    • Use of box-shadow.

Git

  • git checkout = to switch branches
  • git checkout -b = the "-b" lets git know it's a new branch we're creating
  • git add -A = stages a file to record changes to file
  • git push origin main merges main with other branch

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • calling javascript in HTML requires you to call it above the closing body tag.
